# Glossary

- **Account** - A user-owned top-level container, such as bank, wallet, exchange, cash, or other.
- **Subaccount** - A named holding inside an account. Each subaccount has exactly one immutable asset.
- **Asset** - A fiat or crypto catalog entry maintained by the backend.
- **Base asset** - The asset used to express converted totals and analytics values.
- **Balance entry** - An immutable balance snapshot for one subaccount.
- **Atomic amount** - Decimal-safe amount storage as text plus decimals, for example `amount_atomic` and `amount_decimals`.
- **Plan** - The user's entitlement level. Current values are `free` and `pro`.
- **Rates snapshot** - Latest backend-cached USD price rows in `asset_rates_usd`.

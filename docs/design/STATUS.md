# Design prototypes status

The `docs/design/` folder contains exploratory design-system and screen prototypes.

For MVP v2 (rewrite), treat the following as **sources of truth**:
- UX behavior and navigation: `docs/ux/*`
- Backend schema and API contracts: `docs/contracts/*`
- Testable feature acceptance criteria: `docs/features/*`

Prototypes in `docs/design/` may lag behind and are not guaranteed to match the latest MVP v2 screen map.
